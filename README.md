# profile

My personal profile site. Live at [sna4i.github.io/profile](https://sna4i.github.io/profile/).

## Built with

- [Astro](https://astro.build/)
- [Tailwind CSS](https://tailwindcss.com/)

## Development

```bash
npm install
npm run dev
```

Open http://localhost:4321/profile/ in your browser.

## Build

```bash
npm run build
```

Static output is generated in `dist/`.

## Deployment

Automatic deployment to GitHub Pages on push to `main` via GitHub Actions — see `.github/workflows/deploy.yml`.

Content is managed in [`src/data/profile.json`](src/data/profile.json).
